# Welcome to SE Shop API Repository

This repo contains the Web Service API source code for the sales website

## Technologies
- NodeJS
- ExpressJS
- MySQL
- Sequelize

## Features
- CRUD products
- CRUD categories
- Wishlist
- Cart
- CRUD orders
- CRUD users
- Authentication
- Authorization
- CRUD genders
- CRUD user statuses
- CRUD order statuses
- CRUD payment forms
- Redirect API version by HTTP header
- ...
